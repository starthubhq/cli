# Environment Setup

Starthub runs on npm and requires Docker and Wasmtime to be installed in order to function properly.

## Prerequisites

Before you can use Starthub, make sure you have the following installed:

- **npm** - Starthub runs on npm
- **Docker** - Required for containerized execution
- **Wasmtime** - Required for WebAssembly runtime support
