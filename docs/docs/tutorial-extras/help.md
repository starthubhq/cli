# help

Lists the available commands.

## Usage

```bash
starthub --help
```

or

```bash
starthub -h
```

## Description

The `help` command displays a list of all available Starthub CLI commands along with their descriptions and usage information.

## Example Output

The help command will show all available commands such as:
- `init` - Initialize a project
- `publish` - Publish an action
- `start` - Start the server
- `stop` - Stop the server
- `logs` - Show server logs
- `status` - Show server status
- `login` - Authenticate with Starthub
- `logout` - Logout from Starthub
- `reset` - Clear the cache
